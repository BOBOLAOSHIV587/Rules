## 功能
1. 蜂窝网络下显示<br>- 蜂窝数据 ISP  & LTE、SA、NSA 等类型<br>- 本地 IPv4、IPv6 位址<br>- 代理节点 ISP、位置
2. Wi-Fi 下显示<br>- 本地 IPv4、IPv6 地址<br>- 路由器 IPv4、IPv6 地址<br>- 代理节点 ISP、位置
