## 功能
1. 简洁的外观配以绿色象征清洁的图标
2. FlushDNS可以显示当前使用的DNS的延迟情况
3. FlushDNS模块点击右上角的刷新即可清除DNS缓存


**两次更新之间建议 _ 间隔 5 分钟以上_，否则又几率页面缓存文档尚未更新导致更新失败<br>
(检查方式为 Surge -> 脚本 -> 脚本编辑器 -> 载入 -> FlushDNS -> 检查代码是否与网页版本相同**
