**TikTok登陆说明：**

**现阶段TikTok大于37.8.0版本已经禁止MITM。所以只能用37.8.0版本才能用脚本解锁**

**壹：节点质量不好，在21.1.0版本无法登录的方法：**

1、设置好TikTok分流。

2、App Store下载最新版TikTok，拔卡，不开脚本，登录。设置iCloud钥匙串登录（已设置iCloud钥匙串跳过此步骤执行第3步骤）。（卸载此版本TikTok。）

3、安装21.1.0版本，不装手机卡，不开脚本打开。（不卸载软件。）

4、安装37.8.0版本，不装手机卡，不开脚本使用iCloud钥匙串登录（不出现iCloud钥匙串登录，退出后台多试几次）。（不卸载软件。）

5、安装21.1.0版本，先不开脚本打开，退出。再开脚本打开，退出。再安装手机卡打开。（不卸载软件。）

6、安装37.8.0版本，开脚本打开，解锁成功。


**贰：节点质量好，在21.1.0版本成功登录的方法：**

1、设置好TikTok分流。

2、安装21.1.0版本，不装手机卡，不开脚本打开。登录账号。退出。再开脚本打开（查看代理软件是否成功mitm。未成功多退出打开几次），退出。再安装手机卡打开。（不卸载软件。）

3、安装37.8.0版本，打开，解锁成功。
