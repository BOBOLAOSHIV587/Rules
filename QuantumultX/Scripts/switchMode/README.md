## 📦 使用教程

**捷径设置方法**
```
打开捷径 → 右上角+ → 加入动作 → 找到“类别”-“网页” → 找到“取得URL内容” → 把响应正则填入URL → 保存完成.
```

**使用方法一**
```
脚本添加重写 → 打开QX → 运行捷径.
```
