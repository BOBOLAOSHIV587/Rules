# 说明


1、使用版本：11.3.2为测试时最新版-可用。


2、使用方法：打开App点个歌提示听歌时长耗尽或缓冲失败（或正常播放无法切换音质、拖动），更换另一首歌即可，多试几次。


3、QX使用此脚本，正常显示会员标识。（左图）


4、Surge&Loon使用此脚本，无法显示会员标识，同时显示年费听歌时长已用尽。正常显示，正常使用，无视即可。（右图）
